# Bais HaMikdash Readiness Fund (Demo)

A polished, responsive Next.js + TypeScript + Tailwind demo website for a restrained philanthropic initiative centered on a preserved-principal readiness fund.

## Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000`.

## Pages
- `/` Home
- `/about` About the Initiative
- `/approbations` Rabbinic Approbations
- `/donate` Donate
- `/transparency` Fund Transparency
- `/faq` FAQ
- `/contact` Contact

## Data and Extensibility
All content is mock-data driven from `src/data/*`.

Integration placeholders are marked in code comments for:
- real rabbinic letters (PDF/scans)
- live fund NAV and reporting feeds
- donor CRM and contact workflows
- payment processor integration

## Notes
- Financial figures and projections are illustrative only.
- This demo is non-operational and intended for presentation/prototyping.
